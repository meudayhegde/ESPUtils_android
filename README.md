##### <center>Android interface for various utilities using ESP8266/ESP32</center>
<center>(ESP32 not tested yet)</center>

## Features of the ESP Utils Application

### Add/Remove multiple ESP Devices.
* #### Using IP address of the ESP Device
* #### Using scanning
  * ```Scan for all the devices with ESPUtil firmware on the local network (WiFI).```


### Settings for each ESP Device.
* #### Wireless
  * ```Set wireless mode (Wifi / Hotspot).```
  * ```Set SSID and PSK, based on the settings begin wireless on startup.```
* #### Credentials
  * ```Requires Credentials for accessing ESP8266 Features.```
  * ```Options to set/modify credentials.```
* #### Reboot
  * ```Option to restart ESP Device.```
* #### OTA Update
  * ```Install new firmware for ESP Device through the application (no need wor wired installation).```


### GPIO Controller
* #### Configure GPIO Switches for all the available ESP Devices
  * ```All the GPIO pins on the ESP Device can be configured as GPIO Switches on the app.Hardware configurations need to be done manually according to requirement. (using Relay switches/using Infrared sensors etc)```
  * ```GPIO Switches can be used as Home Widgets.```


### IR Remote Controller
```
Note: For this feature to properly work, IR Sender and Receivers need to be properly configured on ESP Device.
```
* #### Import/Export Remote controller configuration
* #### Create new Remote Controller
  * ```Using Infrared Signal Capture method. To use this method, click intended button on IR Remote controller pointing towards ESP Device while listening on android application.```
  * ```Capture all the Buttons from the IR Remote Controller.```
  * ```Use captured buttons as Home Widgets for Quick Access.```
  * ```UI Customization Option for Captured button.```
  * ```When button is clicked, Corresponding IR Signal will be emitted form the IR Sender```


### UI Customization
* #### Light and Dark Material theme.
  * ```By default Follow System Theme (Android 10+)```
  * ```Switch between Light and Dark themes manually.```
* #### Select Home Fragment (ESP Device/ GPIO/ IR Remote)
